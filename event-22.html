Преамбула:

Круглый стол проводится в целях повышения информированности о компетенциях предприятий ОПК, как уже применяемых в совместных проектах, так и потенциально перспективных для топливно-энергетического комплекса. В ходе мероприятия планируется обратить особое внимание на предложения организаций ТЭК по стимулированию процессов диверсификации.

Вопросы для обсуждения:

  1. Раскрытие технологического потенциала ОПК для нужд ТЭК.
  2. Обмен компетенциями предприятий ОПК и ТЭК.
  3. Роль предприятий ОПК в увеличении эффективности ТЭК России.
  4. Тиражирование успешных кейсов по диверсификации ОПК для нужд энергетики.

Дискуссия:

Участникам дискуссии предлагается обсудить следующие вопросы:

  1. Тиражирование успешных кейсов по диверсификации ОПК для нужд энергетики.
  2. Энергоэффективные инструменты для оздоровления экономики.
